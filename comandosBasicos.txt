comandos basicos git 
mkdir + nombreCarpeta , crea carpeta
touch + nombre+extencion del documento , crea documento, eje index.txt

cd + nombre de carpeta ingresas a carpeta 
cd  .. volves una carpeta
ls ves que hay en esa carpeta
ls -la ves incluso los archivos ocultos
diff muestra los cambios antes de hacer git add
pwd ves las ruta en la cual estas parado actualmente
